# SCBB Admin (scbb-admin)

Простой шаблон админки на Vue 3 + Vite (JavaScript). Включает:
- Vue 3 + Vite
- Vue Router
- Pinia (state)
- Element Plus (UI)
- Моковая авторизация (composable)
- ESLint, Prettier, husky, Vitest, GitHub Actions

Быстрый старт (локально):
1. Убедитесь, что у вас установлен Node.js и npm на машине.
2. В корне проекта выполните:

   npm install
   npm run prepare   # установит husky hooks
   npm run dev

Запуск тестов:

   npm run test

Примеры и поведение заготовки:
- Логин: в текущей версии используется моковая авторизация — можно ввести любой логин/пароль, и вы попадёте в админку (всё хранится в localStorage).
- Компоненты UI: использую Element Plus — готовые кнопки, формы и карточки.

Если в этой среде npm недоступен (как в моей попытке), я подготовил все файлы и конфигурации; запусти команды выше на своей локальной машине.
